# Release Plan

## Alpha Readiness

- Core auth, RBAC, tenants, ingestion, search, alerting, incidents, and agent policy APIs run locally.
- Route contract tests, Docker Compose config, frontend build, and backend compile pass.
- Product roadmap and architecture docs exist.

## Beta Readiness

- End-to-end ingestion to normalized searchable events.
- Parser catalog and parser tests.
- Detection rules with MITRE mapping and simulation.
- Audit trail API and security review.
- Analyst workflows for search, alerts, cases, dashboards, and agent management.
- Deployment, backup/restore, and admin docs.

## Commercial Release Readiness

- Multi-tenant RBAC hardened.
- Performance baseline and load test evidence.
- Compliance reports and report scheduler.
- AI SOC features with evidence references, confidence, audit logs, and provider settings.
- Release checklist, known issues, changelog, rollback notes, and support docs complete.
