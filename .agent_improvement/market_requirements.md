# Market Requirements

This tracker maps Inso SIEM toward modern commercial SIEM/SOC expectations: multi-source ingestion, custom forwarders, parsing and normalization, high-speed search, dashboards, alerting, correlation, AI-assisted investigation, case management, SOAR, auditability, multi-tenancy, compliance reporting, and enterprise security controls.

## Baseline Requirements

- Collect data from agents, APIs, syslog, cloud services, files, containers, identity systems, and SaaS tools.
- Normalize events into a consistent schema and preserve original evidence.
- Provide fast search with saved searches, query history, field discovery, timeline views, aggregation, and dashboards.
- Support detection-as-code, correlation, MITRE ATT&CK mapping, alert triage, cases, evidence, and response workflow.
- Provide tenant-safe RBAC, MFA, session management, audit trails, retention controls, data masking, and usage metering.
- Provide AI safely: evidence-based summaries, visible generated queries before execution, provider configuration, usage metering, and audit logs.
